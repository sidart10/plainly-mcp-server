## Description

### What I Need

I need an **After Effects template** that animates an AI Agent workflow diagram. This will be used with Plainly Videos API to create unlimited variations automatically.

### The Design

Clean, minimalist, technical aesthetic:
- **Background**: Pure black
- **Style**: Blue boxes with white borders
- **Arrows**: Hand-drawn style showing request/response flow
- **Text**: White, casual/handwritten font

**See attached image** for the exact design to recreate and animate.

### Animation Requirements

**Sequence (15-20 seconds)**:
1. Elements fade in sequentially
2. Arrows draw with "write-on" effect
3. Particles travel along arrows showing the loop
4. Smooth, professional transitions

**Style Reference**: Similar to Fireship, ThePrimeagen, or tech explainer videos

### Technical Requirements

**Must Have**:
- ✅ Dynamic text parameters (20+ fields)
- ✅ Color controls (6 color parameters)
- ✅ Animation speed controls
- ✅ Works with Plainly Videos platform
- ✅ Clean, well-organized project file

**Specifications**:
- Resolution: 1920x1080 (16:9) AND 1080x1080 (1:1)
- Duration: 15-20 seconds
- Frame rate: 30fps
- Format: .aep (After Effects project)

### Deliverables

1. **.aep project file** (After Effects, latest version)
2. **Both aspect ratios** (16:9 and 1:1)
3. **All assets included** (fonts, any images)
4. **Parameter documentation** (what can be changed)
5. **Test render video** (1080p MP4 showing animation)
6. **Quick setup guide** (how to customize)

### Dynamic Parameters Needed

**Text (can be changed per video)**:
- Title
- AI Agent label
- Model label
- Tools label
- Request label
- Response label
- Company name
- Footer text

**Colors (can be changed)**:
- Primary color (boxes)
- Background color
- Text color
- Arrow color
- Glow/accent color

**Animation**:
- Speed control (slow/normal/fast)
- Number of loop cycles
- Particle toggle (on/off)
- Glow toggle (on/off)

### What Success Looks Like

I should be able to:
1. Open the .aep file
2. Change text in a composition
3. Change colors via color controls
4. Export and get perfect results
5. Upload to Plainly and render via API

### Budget & Timeline

- **Timeline**: 5-7 days
- **Revisions**: 2 rounds included
- **Rush available**: +$100 for 3-day delivery
